# Configuration anchors

Place the Central Server **global configuration anchor** here (e.g. `TL-TEST-anchor.xml`), downloaded after
initializing the Central Server (README Step 2). Anchors are environment-specific and are **gitignored**
(`*.xml`); only this README is tracked. `xroad/config/xrdsst-config.yaml` references `./xroad/anchors/TL-TEST-anchor.xml`.
